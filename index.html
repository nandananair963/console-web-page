WHAT IS VISUAL STUDIO CODE?

Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

HOW TO SET IT UP?

1.Download and install VS Code.
2.Create a new file
3.See an overview of the user interface.
4.Install support for your favorite programming language.
5.Change your keyboard shortcuts and easily migrate from other editors using keymap extensions.
6.Customize your editor with themes.
7.Explore VS Code features in the Interactive Editor Playground.

WHAT IS A CONSOLE SCREEN?

Windows Console is the infrastructure for console applications in Microsoft Windows. An instance of a Windows Console has a screen buffer and an input buffer. It allows console apps to run inside a window or in hardware text mode.
